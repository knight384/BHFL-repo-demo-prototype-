# BFHL Full Stack Project

## 📌 Overview
This project is a full-stack application that processes hierarchical graph data.
It takes relationships as input and builds structured tree hierarchies.

---

## 🚀 Features
- Input validation
- Duplicate edge detection
- Tree construction
- Cycle detection
- Depth calculation
- Summary generation

---

## 🛠️ Tech Stack

### Backend
- Node.js
- Express.js

### Frontend
- HTML
- CSS
- JavaScript

---

## 🔗 API Endpoint

POST /bfhl

### Request
{
  "data": ["A->B", "A->C", "B->D"]
}

---

## ▶️ Run Locally

1. Install dependencies
npm install

2. Start server
npm start

3. Open frontend/index.html in browser
